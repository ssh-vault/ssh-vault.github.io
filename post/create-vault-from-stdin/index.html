Create vault from stdin


ssh-vault can read from standard input (stdin) when creating a vault.

Instead of creating a vault from scratch, a vault can be created by using an existing file, for example if Alice wants to send a file to Bob she could so something like:

cat /path/to/file | ssh-vault -u bob create bobs.vault

you can choose any name you want for the vault, in this example the vault contents will be written to file bobs.vault
