What is ssh-vault?

A command line tool (cli) for creating encrypted files “vaults” using ssh rsa keys for encrypt/decrypt.

A vault password is encrypted using the public ssh key of the receiver and the main data of the vault using AES256, in order to see/decrypt the vault contentes, the receiver will need to use his private key & password in case of needed to be available to edit/view the contents.

You can learn more about generaring an SSH key here: https://help.github.com/articles/generating-an-ssh-key/

Why ?

Because of the need to share sensitive data without the need to share a password.
